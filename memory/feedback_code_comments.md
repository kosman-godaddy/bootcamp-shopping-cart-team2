When adding comments to code in this project, end them with ` - Ian` (e.g. `// Expandable dropdown for product details - Ian`).

**Why:** The team attributes code to individual authors using a dash-name suffix in comments. Ian is the user's name. Other team members use the same pattern (e.g. `-Ahmed`, `-Nyla`).

**How to apply:** Any comment added to this codebase should end with ` - Ian` unless it's clearly authored by someone else or the comment pre-exists without attribution.
